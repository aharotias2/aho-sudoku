### 実行方法

| オプション | 短かいオプション | 説明                                              |
|------------|------------------|---------------------------------------------------|
| theme      | t                | light, dark                                       |
| help       | h                | ヘルプを表示                                      |
| version    | v                | バージョンを表示                                  |
| debug      | d                | デバッグモードで起動 (デバッグボタンが表示される) |
